Important FAQ

What is the Document Object Model (DOM)?

The Dom is programming interface for web documents.

How do you select an element from the DOM?

By using getElementById,querySelectorAll,querySelector,getElementsByClassName and getElementsByTagName

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is particularly useful for handling events on dynamically added elements.Instead of attaching a click handler to each item, event delegation allows you to attach a single listener to the parent container.
